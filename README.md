## Installation

Install Dependencies for next-frontend

```bash
>>> cd next-frontend

>>> npm install
# or
>>> yarn
```

Then, run the development server:

```bash
>>> cd server

>>> yarn watch # for typescript

>>> npm run dev
# or
>>> yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
